    Running simulation from Intellij

    Intellij will ignore % Provided library dependencies by default. However, this means it cannot find the Spark libraries, since these have to be tagged as such, because these are assumed to be supplied by the Spark server. To run a simulation from within Intellij, you must edit the run configuration and under "Configuration" check the option "Include dependencies with Provided scope".
